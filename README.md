# Web App IDE Ayuntamiento de Almonte

Web App desarrollada en **React** con **API** backend, creada para el **Ayuntamiento de Almonte**. La aplicación sirve como plataforma del **IDE del Ayuntamiento**, permitiendo el almacenamiento y gestión de **enlaces e imágenes** referenciadas a las distintas webs del ayuntamiento.

## Características

- Interfaz web interactiva desarrollada con **React**.
- Sistema de almacenamiento y gestión de **links e imágenes**.
- Integración con **API** para la gestión centralizada de contenidos.
- Optimizada para uso interno del **Ayuntamiento de Almonte**.
